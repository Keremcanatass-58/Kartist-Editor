• BCrypt: Adaptif hash fonksiyonu ⟶ MD5/SHA256'dan çok daha güvenli şifre saklama.

Sprint 1 — Güvenlik Altyapısı

Sprint 1
Siber Güvenlik Katmanları
Tamamlandı

Sistemin güvenlik temellerinin atılması ve OWASP Top 10 standartlarına uyum.

  • SECURITY HEADERS X-XSS-Protection, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy ve Content-Security-Policy header'ları özel middleware ile her HTTP yanıtına eklendi.
  • BCRYPT HASHING Düz metin (plaintext) saklanan tüm mevcut şifreler BCrypt adaptif hash algoritmasına migrate edildi. Work factor: 12.
  • CSRF KORUMALARI Tüm POST formlarına Anti-Forgery Token eklendi. Global [ValidateAntiForgeryToken] filtresi aktif.
  • RATE LIMITING Brute-force saldırılarına karşı IP bazlı istek sınırlama middleware'i devreye alındı.
  • SQL INJECTION ÖNLEME Tüm Dapper sorguları parametrik hale getirildi. String concatenation ile sorgu oluşturma tamamen kaldırıldı.
  • INPUT VALIDATOR Kullanıcı girdileri için merkezi InputValidator sınıfı güçlendirildi: XSS, path traversal ve injection kontrolü.

SecurityHeadersMiddleware.cs — Kod Örneği

// Her HTTP yanıtına güvenlik header'ları ekleyen middleware public async Task InvokeAsync(HttpContext context) { context.Response.Headers.Append("X-XSS-Protection", "1; mode=block"); context.Response.Headers.Append("X-Frame-Options", "SAMEORIGIN"); context.Response.Headers.Append("X-Content-Type-Options", "nosniff"); context.Response.Headers.Append("Referrer-Policy", "strict-origin-when-cross-origin"); // Content-Security-Policy (CSP) ile kaynak kısıtlamaları string csp = "default-src 'self'; script-src 'self' ..."; context.Response.Headers.Append("Content-Security-Policy", csp); // Sunucu parmak izini gizle context.Response.Headers.Remove("Server"); context.Response.Headers.Remove("X-Powered-By"); }

Sprint 3 — AI Entegrasyonu

Sprint 3
Yapay Zekâ Motoru
Tamamlandı

Groq LLM ve görsel üretim API'lerinin platforma entegrasyonu ve akıllı fallback sistemi.

  • GROQ API (LLM) Kullanıcının tasarım isteğini doğal dilde anlayan ve kart içeriği, renk paleti, font önerisi üreten yapay zekâ servisi entegre edildi.
  • AI GÖRSEL ÜRETİMİ Metin açıklamasından (text-to-image) otomatik arka plan görseli üreten Pollinations.ai entegrasyonu tamamlandı.
  • FALLBACK ALGORİTMASI Ana API çökme/kota durumlarında isteklerin otomatik olarak Pollinations → Picsum zincirine yönlendirildiği 3 katmanlı yedekleme sistemi.
  • PROMPT MÜHENDİSLİĞİ Türkçe kullanıcı girdilerinin yüksek kaliteli İngilizce prompt'lara dönüştürüldüğü EnhancePrompt algoritması geliştirildi.

AiImageService.cs — Fallback Mekanizması

// 3 Katmanlı AI Görsel Üretim Fallback Zinciri public async Task<AiImageResponse> GenerateImageAsync( string prompt, CancellationToken ct) { // Katman 1: Ana Provider (OpenAI veya Pollinations) var result = await TryGenerateWithPollinationsAsync(prompt, ct); if (result.Success) return result; // Katman 2: Keyword-based Fallback (Picsum) var fallback = await TryGenerateWithKeywordFallbackAsync(prompt, ct); fallback.Warnings.InsertRange(0, result.Warnings); return fallback; } // Türkçe → İngilizce prompt zenginleştirme private string EnhancePrompt(string prompt) { if (prompt.Contains("doğum günü")) return "festive birthday celebration, colorful bokeh..."; return $"{prompt}, professional card background, aesthetic"; }
